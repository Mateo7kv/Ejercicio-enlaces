Bienvenido a la Página 3

Llegaste a la tercera y última página del sitio. Aquí encontrarás un resumen de lo que hemos visto sobre HTML y cómo se conectan los conceptos para construir un sitio web completo.

¿Qué hemos aprendido?

En la Página 1 vimos que HTML es el lenguaje base de la web. Aprendimos que todo documento necesita una estructura con <html>, <head> y <body>.

En la Página 2 repasamos las etiquetas de formato como <strong>, <em>, <mark>, <del> e <ins>, que enriquecen el contenido y le dan significado al texto.

En esta Página 3 confirmamos que los enlaces son fundamentales para conectar páginas entre sí, usando la etiqueta <a href="..."> con la ruta del archivo destino.

💡 Conclusión: HTML, los comentarios, el cambio de colores y los enlaces son herramientas esenciales para construir sitios web organizados, legibles y bien estructurados.
