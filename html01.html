Bienvenido a la Página 1

Hola mi nombre es Carlos Mate Peña y aqui veremos algunas de las cosas aprendidas en clase de programación web HTML, el lenguaje que da estructura a todas las páginas web. HTML significa HyperText Markup Language y es la base del desarrollo web.

Con HTML podemos crear encabezados, párrafos, imágenes, listas y, por supuesto, enlaces que conectan unas páginas con otras. Navega usando los botones de abajo para ir a la siguiente página.

Para el diseño de esta actividiad decidí que dada pagina tuviera su propio color y contenido, pero las 3 estan conectadas entre sí.
